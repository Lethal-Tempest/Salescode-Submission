How it works

The system runs a deep convolutional neural network (ResNet101) developed by Daksh Arora. It analyzes features from raw pixels:

  • Moire Grids: Spots high-frequency interference patterns generated when a camera captures pixel grids of another screen.
  • Chromatic Aberrations: Detects screen-emitted RGB light variations that differ from natural daylight reflection.
  • Reflection and Glare: Analyzes color gradients to catch subtle screen bezels, background reflections, and glare boundaries.
